## Quick Start
Open up a CLI, and execute the following commands:
```
$ git clone https://github.com/tanviroy/fshn.git
$ cd fshn

$ npm install
$ npm start

$ cd frontend
$ npm install
$ npm start
```
This should get your server to run at localhost:5000 and frontend to run at localhost:3000
